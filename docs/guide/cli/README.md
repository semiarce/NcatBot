# CLI 工具 — 命令行管理 NcatBot

> 通过 `ncatbot` 命令完成项目初始化、启动、插件管理、配置管理和 NapCat 诊断。

## Quick Start

安装 NcatBot 后即可使用 `ncatbot` 命令：

```bash
# 初始化项目（生成 config.yaml + plugins/ + 模板插件）
ncatbot init

# 启动 Bot
ncatbot run

# 以开发模式启动（debug + 热重载）
ncatbot dev

# 进入交互模式（REPL）
ncatbot
```
